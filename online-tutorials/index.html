Introducción al desarrollo de software libre

El objetivo de este tutorial es que los asistentes puedan contribuir con confianza a proyectos de software libre, sea cual sea su nivel técnico.

La sesión será práctica, se empezará desde cero, y se guiará a los participantes en todos los pasos necesarios para contribuir a proyectos de software libre. Desde el punto de vista técnico, pero también humano.

Acceder

El evento se realizará usando la plataforma de videoconferencia meet.jit.si. El enlace al evento se publicará unos minutos antes de empezar.

En parallelo, disponemos del group de Telegram Intro open source que ya está activo. El canal está pensado para hacer preguntas, debatir... antes, durante, y después del evento. Se recomienda unirse, ya que también se usará para comunicarse en caso de problemas técnicos durante el evento.

Contenidos

Los contenidos principales que se van a ver son:

  • Git / GitHub
  • Integración Continua (CI)
  • Best practices en Python
  • Trabajo en equipo
  • Inclusión y diversidad

Requisitos

No hay requisitos mínimos, pero conocimientos básicos de git (git add, git commit, git push...), y de Python serán útiles. Se debería disponer de git instalado para poder seguir el tutorial de forma práctica. Así como de un editor de texto apropiado para programar.

Formato

El tutorial se realizará online, usando jitsi. No es necesario registrarse ni instalar software, aunque para dispositivos móviles es recomendable instalar la aplicación.
